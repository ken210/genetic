# Genetic Algorithm Playground
ken210's first attempt on Genetic Algorithms.

## Install
```Shell
$ git clone git@github.com:ken210/genetic
$ npm install
```

## Build
```Shell
$ ./node_modules/.bin/babel src --out-dir lib --source-maps
```

## Usage
```Shell
$ node lib/index.js [task]
```

## Tasks
- `run-normal` (default)
- `find-array`
